# QuarterFull 📚

あらゆる学習を1つにまとめる、シンプルな暗記(フラッシュカード)学習アプリです。
ブラウザだけで動作するため、インストール不要。PC・スマートフォン・タブレットからいつでも学習できます。

## ✨ 特徴

- 📖 デッキごとのフラッシュカード学習
- 🧠 間隔反復による効率的な復習
- 📊 学習履歴・正答率の記録
- 📁 CSVによるカードの一括インポート・エクスポート
- 🔊 英単語・例文の音声読み上げ
- 🌙 ライト／ダークテーマ
- 💾 自動保存（localStorage）
- 📱 PWA対応（ホーム画面に追加可能）
- 🌐 GitHub Pagesでそのまま公開可能


## 🚀 使用技術

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage
- Web Speech API
- Service Worker (PWA)
